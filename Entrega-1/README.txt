README - Ximena's Portfolio (HTML Project)

Este proyecto es una página web estática desarrollada únicamente con HTML y CSS embebido. Tiene como objetivo practicar y demostrar el uso de etiquetas semánticas, estructura básica de una web y recursos externos como íconos.

Estructura del Código:

- <!DOCTYPE html>: Define el tipo de documento como HTML5.

- <head>: Contiene metadatos, título, descripción, autor y un enlace a la librería de íconos Font Awesome.

- <style>: Incluye estilos CSS simples directamente en el archivo HTML.

- <body>: Dividido en:

	- <header>: Título de la página.

	- <nav>: Menú de navegación con íconos.

	- <main>: Secciones internas (about-me, projects, contact).

	- <footer>: Enlace al perfil de GitHub.


Características del Código:

- Etiquetas semánticas (header, nav, section, footer).

- Menú de navegación.

- Íconos integrados mediante Font Awesome.

- Tabla para organización de proyectos.

- Listas no organizada de skills

- Formulario HTML funcional sin JavaScript.

- Estilos en línea y embebidos, sin archivos CSS externos.

- Links.


Requisitos:


- Navegador web moderno.

- No necesita conexión a internet (salvo para íconos e imagenes).

- No requiere servidor ni backend es local.
